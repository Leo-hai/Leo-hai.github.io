字节流：InputStream , OutputStream

字符流：Reader , Writer
操作文件：创建文件，并写入数据 FileWriter。

1. FileWriter fw = new FileWriter("demo.txt");     // 创建文件

2. // 写入数据
    fw.write("abcde");
3. // 将数据刷新到目的地中。
    fw.flush
// 或 fw.close
// flush刷新后，流可继续使用。close刷新后，会将流关闭。

文本文件读取

// 创建一个文件流地区对象，如文件不存在，会抛异常
FileReader fr = new FileReader("demo.txt");
// 读
第一种方式
int ch = fr.read();
// read,一次读取一个字符，而且自动往下读。
第二种方式
定义一个字符数组。用于存储读到的字符个数。

char[] buf = new char[1024];
int num = fr.read(buf);
