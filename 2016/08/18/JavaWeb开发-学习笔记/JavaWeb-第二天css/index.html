JavaWeb-第二天css

Css

css的简介

Cascading Style Sheets 层叠样式表
层叠：一层一层的
样式表：很多的属性和属性值
使页面显示效果更加好。

css和html的结合方式（四种）

（1）在每个html标签上面都有一个属性 style，把css和html结合在一起。
（2）使用html的一个标签实现<style>，写在head里面
        <style type="text/css">
            css代码;
        <style>
（3）在<style>标签里面 使用语句
        @import url(css文件的路径)
（4）使用头标签link，引入外部css文件。
        <link rel="stylesheet" type="text/css" href="css文件路径"/>

一般都使用第四种结合方式

css的优先级和选择器

由上到下，由内到外。优先级由低到高。
后加载的优先级高。

选择器：（三种）
（1）标签选择器：使用标签名作为选择器的名称
（2）class选择器：每个html标签都有一个属性class  加.
（3）id选择器：每个标签上面都有一个属相id  加#

选择器优先级：
style > id选择器 > class选择器 > 标签选择器

css扩展选择器

（1）关联选择器
        嵌套标签里的样式
        标签之间空格隔开 div p {}
（2）组合选择器
        把不同的标签设置成相同的样式
        逗号隔开         div,p {}
（3）伪元素选择器
        css里面定义好的元素，直接拿过来使用 a:link
        原始状态        :link
        鼠标停留状态    :hover
        点击            :active
        点击之后        :visited

css的盒子模型

（1）边框
    border 粗细 样式 颜色
    border-top
    border-bottom
    border-left
    border-right
（2）内边距
    padding 距离
    padding-top ...
（3）外边距
    margin 距离
    margin-top ...

position
    - absolute 将对象从文档流中拖出，可以使用top等属性进行定位
    - relative 不会把对象从文档流中拖出，也可以进行定位
