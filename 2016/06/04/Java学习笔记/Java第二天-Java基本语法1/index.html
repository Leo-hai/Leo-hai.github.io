Java第二天-Java基本语法1

Java语言基本组成

关键字

标识符

  • 在程序中自定义的一些名称
  • 由26个英文字母大小写，数字：0-9符号:_$组成

  • 定义合法标识符规则：

    数字不可以开头。
    不可以使用关键字

  • Java中严格区分大小写

    Java中的名称规范

  • 包名：多单词组成时所有字母小写。xyz
  • 类名接口名：多单词组成时，所有单词首字母大写。XxYyZz
  • 变量名和函数名：多单词组成时，首字母小写，后面首字母大写。xxxYyyZzz
  • 常量名：所有字母都大写。多单词时每个单词都用下划线连接。XXX_YYY_ZZZ

常量和变量

每个字节有8位（二进制位）
十进制–>二进制 ：除以2取余数
二进制–>十进制 ：乘以2的幂数
十进制–>十六进制 ：先转成二进制，再转成十六进制

负数的二进制表现形式：二进制取反 再 + 1
取反（将二进制的1变成0，0变成1） 负数的最高位都是1

变量

变量：就是将不确定的数据进行存储。也就是需要在内存中开辟一个空间。

数值型：

整数类型
byte（-2^7~127）
short()
int(2^31)
long()

浮点类型
float
double

字符型

char

布尔型

boolean

运算符

循环结构

while， do-while， for

break：跳出循环
continue：只能作用于循环结构。（继续循环）结束本次循环，继续下一次循环。
