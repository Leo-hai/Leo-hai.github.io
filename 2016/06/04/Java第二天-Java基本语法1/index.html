Java第二天-Java基本语法1

Java语言基本组成

关键字

标识符

  • 在程序中自定义的一些名称
  • 由26个英文字母大小写，数字：0-9符号:_$组成

  • 定义合法标识符规则：

    数字不可以开头。
    不可以使用关键字

  • Java中严格区分大小写

    Java中的名称规范

  • 包名：多单词组成时所有字母小写。xyz
  • 类名接口名：多单词组成时，所有单词首字母大写。XxYyZz
  • 变量名和函数名：多单词组成时，首字母小写，后面首字母大写。xxxYyyZzz
  • 常量名：所有字母都大写。多单词时每个单词都用下划线连接。XXX_YYY_ZZZ

常量和变量

每个字节有8位（二进制位）
十进制–>二进制 ：除以2取余数
二进制–>十进制 ：乘以2的幂数
十进制–>十六进制 ：先转成二进制，再转成十六进制
