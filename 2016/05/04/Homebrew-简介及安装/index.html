Homebrew简称 brew，是 Mac OSX 上的软件管理工具，能在 Mac 中方便的安装或者卸载软件。

Homebrew 安装打开终端，输入下面的命令即可进行安装

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装过程如下图：

由于本人在安装时未截图，故在网上找了张图放上。

HomeBrew 使用

搜索软件：

brew search 软件名

安装软件：

brew install 软件名

卸载软件：

brew remove 软件名

更多的，还请查看官网：http://brew.sh/index_zh-cn.html
